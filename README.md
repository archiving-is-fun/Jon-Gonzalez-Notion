## Archivo del Notion de Jon Gonzalez

Debido a que Jon Gonzalez eliminó su cuenta, he decidido realizar un archivo de su Notion guardando las imágenes y el contenido escrito de cada página, con el objetivo de preservar la información en caso de que el Notion original o sus publicaciones terminen desapareciendo por completo.

## Contribuciones

Si alguien dispone de alguna gráfica o contenido de Jon que todavía no se encuentre en este repositorio, puede abrir un Pull Request.

Por favor, intentad respetar el formato actual:

- Cada gráfica debe ir dentro de su propia carpeta
- Utilizar el formato `TituloGrafica`
- Si Jon añadió texto o explicaciones, incluirlas en un archivo `.txt` o `.md`

Revisaré periódicamente las Pull Requests y aceptaré aquellas que mantengan la organización del repositorio.
